│ Layer 1: Holographic Memory Storage                      │

## Quick Start

```bash
# Install
pip install -r requirements.txt

# Run core
python -m prometheus.core

# Query
python -m prometheus.query "Write a Python function to sort a list"
```

## Components

- `/memory` - Holographic knowledge storage
- `/reasoning` - Fractal recursive engine
- `/swarm` - Specialist network
- `/navigation` - Internet semantic search
- `/interface` - User interaction layer
