# Copilot Instructions for OpenForge

This is an OpenForge-generated Next.js 14 application.

## Tech Stack
- Next.js 14 (App Router)
- React 18 with TypeScript
- Tailwind CSS for styling
- shadcn/ui components
- Lucide React for icons

## Code Style Guidelines
- Use TypeScript for all new files
- Prefer functional components with hooks
- Use Tailwind utility classes for styling
- Follow the existing component patterns
- Use Lucide icons: `import { IconName } from "lucide-react"`

## File Structure
- Pages go in `src/app/`
- Components go in `src/components/`
- Utils go in `src/lib/`
- Types should be defined inline or in dedicated files

## State Management
- Use React useState for local state
- Use React useEffect for side effects
- Prefer server components where possible
- Keep client components minimal
